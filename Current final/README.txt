---English:-------------------------------------------
The .exe is made with launch4j.
It is available on: http://launch4j.sourceforge.net/

settings suggested:
  Basic:
    Output file:  select path for .exe file
    Jar:          input file
    Icon:         select your folder where NEMA is and go to Daten\Nema.ico
  JRE:
    Min JRE version:  1.6.0

Press "Build wrapper"



While running NEMA.exe under Windows there are 2 requirements:
1) Make sure that in the same place as NEMA.exe is also a folder "daten" with both .pdf inside.
 yourfolder\NEMA.exe
 yourfolder\daten\de_NEMA.pdf
 yourfolder\daten\en_NEMA.pdf
2) Have JavaRuntimeEnvironment 1.6.0 or higher installed


---Deutsch:-------------------------------------------
Die .exe ist erstellt mit launch4j.
Dieses Projekt ist verfuegbar unter: http://launch4j.sourceforge.net/

vorgeschlagene Einstellungen:
  Basic:
    Output file:  Waehle den Ort der Ausgabe der .exe-Datei
    Jar:          Eingabedatei
    Icon:         gehe zum Ordner "NEMA" und waehle dort "Daten\Nema.ico"
  JRE:
    Min JRE version:  1.6.0

Druecke abschliessend "Build wrapper" zum Erstellen der .exe.



Um NEMA.exe unter Windows korrekt auszufuehren muessen 2 Bedingungen erfuellt sein!
1) Stelle sicher, dass im selben Ordner wie die NEMA.exe ebenfalls der Ordner "daten" mit beiden .pdf darin vorhanden ist.
 deinOrdner\NEMA.exe
 deinOrdner\daten\de_NEMA.pdf
 deinOrdner\daten\en_NEMA.pdf
2) Es muss die JavaRuntimeEnvironment 1.6.0 oder neuer installiert sein.
